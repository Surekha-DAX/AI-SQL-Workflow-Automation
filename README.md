This project demonstrates how I designed and implemented an AI-driven workflow that automates SQL query execution using natural language input.

## 🚀 Features
- **Trigger-based orchestration**: Workflow starts when a chat message is received.
- **AI Agent Integration**: Google Gemini Chat Model processes natural language queries.
- **Memory Management**: Simple Memory node maintains query context.
- **Database Execution**: Postgres node executes SQL queries seamlessly.
- **Reusable Subflows**: Modular design with `sub_sql_exe` for scalable execution.
